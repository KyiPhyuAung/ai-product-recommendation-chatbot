# 🛒 AI Product Recommendation Chatbot

This project is an AI-based product recommendation system built using Amazon product dataset.

## Features
- Product search using natural language
- Intelligent recommendation system
- TF-IDF + Nearest Neighbor model
- Intent detection (audio, phone, laptop, etc.)
- Streamlit chatbot UI

## Technologies
- Python
- Pandas
- Scikit-learn
- Streamlit
